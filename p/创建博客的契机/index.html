创建博客的契机

博客是什么，为什么我要建立，以及怎么建立

2024-12-19

介绍

博客起源于家用计算机和互联网的普及。最初，人们借助博客平台来表达个人观点、分享思想。然而，随着移动设备的普及和技术门槛的提升，传统的博客平台逐渐被淘汰，转而成为一小部分人记录个人学习和感悟的载体。

科技或知识的进步，往往源于人们的好奇心和实际需求。在不断实践的过程中，我们累积经验、进行量化分析、得出结论，然后不断反思和迭代。我创建博客的目的，就是希望在某个阶段能够记录下我的思考，并将这些思考系统化地整理出来。

我的思考

在我看来，人生中并没有一份“最佳说明书”，而是需要我们自己去摸索、去调整、去优化。另一方面，一个合格的说明书，既需要细节的步骤，也需要提供相应的工具，来帮助我们实现目标。

工具准备：

  1. 一台电脑
  2. Git 程序
  3. GitHub 账号
  4. VPN（科学上网工具）
  5. Hugo 程序

创建博客的步骤：

  1. 建立 GitHub 账号
    在 GitHub 上创建个人公开仓库，仓库名称为 username.github.io。在仓库设置中，启用 GitHub Pages，获得你自己的网页链接。

  2. 下载 Hugo 主题
    选择一个合适的 Hugo 主题，并阅读文档文件，修改主题以符合个人需求。

  3. 处理报错
    在开发过程中，你可能会遇到一些错误或问题。你可以使用国内的豆包工具、国外的 GPT，或者微软的 Copilot 来帮助你调试和修改代码。

  4. 参考视频教程
    具体的操作可以参考以下视频教程：Bilibili—学习视频

测试

博客blog,历史起源于家用计算机和互联网的普及，大家尝试去建立网站表达个人的观念。后来，由于移动手机平台兴起，繁琐以及部分技术门槛的阻挠下，逐渐落寞。 成为了部分人们，记录个人学习感悟的载体。

科技或者知识的大多数起源于好奇和需求，在鼓捣鼓捣实践的过程中累积经验并保留，严谨的数学量化后，人文分析定个结论，继续循环往复。而我创建博客正希望 能够记录我人生某个阶段的部分的思考，作为回忆。同时系统性整合我的想法。

我认为没有最好的说明书，只有最好的售后。没有标准又陷入了诡辩。那么我认为一个合格的说明书：既要有细节的步骤，又要提供相应的工具。eg:最好还要有售后 工具：0.一台电脑 1.git程序 2.github账号 3.vpn（科学上网的梯子）4.hugo程序 步骤：1.建立GitHub账号 ，创建个人公开仓库，仓库名为username.github.io,仓库设置内点击page,链接即为你的网页。 2.下载hugo的主题，并阅读document文件修改个人主题。 3.处理报错，可以使用国内用豆包/国外gpt或者微软的copilot去咨询具体修改。 4.以上具体操作可参考的视频url:https://www.bilibili.com/video/BV1bovfeaEtQ
